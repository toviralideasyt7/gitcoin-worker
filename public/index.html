Gitcoin Passport API Proxy

This is a Cloudflare Worker that proxies requests to the Gitcoin Passport API.

Usage

To use this API, make a GET request to:

/api/gitcoin/0xYOUR_ADDRESS

Example:

/api/gitcoin/0x1234567890abcdef1234567890abcdef12345678

Response Format

The API returns Gitcoin Passport score information in the following format:

{
  "address": "0x1234...",
  "score": "25.1",
  "passing_score": true,
  "last_score_timestamp": "2023-09-01T12:00:00Z",
  "expiration_timestamp": "2023-10-01T12:00:00Z",
  "threshold": "20",
  "stamps": {
    "StampName1": {
      "score": "5.0",
      "dedup": true,
      "expiration_date": "2023-10-01T12:00:00Z"
    },
    "StampName2": {
      "score": "10.0",
      "dedup": true,
      "expiration_date": "2023-10-01T12:00:00Z"
    }
    // ... more stamps
  }
}
